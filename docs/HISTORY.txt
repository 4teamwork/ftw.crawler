- Add crawler_site_id option for improving purging. [jone]

1.3.0 (2017-11-03)
------------------

  when you set the site url to a sitemap url. Please use the "sitemap_urls"
  attribute instead. You also need to purge the Solr index manually and reindex.
  [jone]
